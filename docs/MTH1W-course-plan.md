# MTH1W — Grade 9 Mathematics (De-streamed) · Full Course Plan

Aligned to **The Ontario Curriculum, Grade 9: Mathematics, 2021 (MTH1W)**.
Strands: A (Mathematical Thinking — woven throughout), B Number, C Algebra,
D Data, E Geometry & Measurement, F Financial Literacy.

**Every subject below includes the same 4 deliverables:**
1. 🌐 **Web lesson** — interactive (LaTeX, figures, graphs, animations, callouts)
2. 📄 **PDF lesson** — downloadable printable version
3. 📝 **3 quizzes** — Easy, Medium, Hard
4. ✅ **Assignment**

> Note: "PDF lesson" needs a small feature add — a PDF attach/link on each lesson
> (see "Build notes" at the end). Everything else is supported today.
